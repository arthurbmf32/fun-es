**GRÁFICO FUNÇÕES SENO E COSSENO**

Projeto desenvolvido por Arthur Bastos e Davi de Melo para a disciplina de Matemática Elementar do curso de graduação em Sistemas de Informação da UFMS.

Objetivo do projeto é visualizar de forma gráfica o comportamento dessas funções ao inserir diferentes valores para os coeficientes das funções.
